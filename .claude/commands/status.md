# 進捗状況

現在の執筆進捗を表示してください。

## 手順

1. `progress.yml` を読み込む
2. `book-config.yml` からタイトルと章数を読む
3. 以下の情報を整理して表示する

## 表示フォーマット

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📖 《タイトル》 執筆進捗
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ ステータス: {status}

■ アウトライン: {✅ 完了 / ⬜ 未着手}

■ 執筆進捗:
  はじめに    {✅ / 🔄 執筆中(2/3セクション) / ⬜}
  第1章       {✅ / 🔄 / ⬜}  10,234字
  第2章       {✅ / 🔄 / ⬜}   9,876字
  ...
  おわりに    {✅ / 🔄 / ⬜}
  ──────────
  合計: {N}/{Total}章 完了  {合計文字数}字

■ レビュー:
  ラウンド: {current_round} / {max_rounds}
  章別: {N}/{Total}章 レビュー済み
  全体チェック: {✅ / ⬜}
  判定: {pass / needs_revision / - }

■ 次のステップ: /project:{次のコマンド}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 文字数の取得

`output/drafts/` 内の既存ファイルからMarkdown記法を除いた本文テキストの文字数をカウントする。
